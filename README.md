# Quotes of the Day App 📖

The "Quotes of the Day" App is an Android application developed using Kotlin that provides users with daily inspirational quotes through a clean and intuitive interface.

---

## ✨ Features

* Displays daily inspirational quotes
* Allows users to save quotes as favorites
* Enables easy sharing of quotes
* Simple, clean, and user-friendly UI

---

## 🛠️ Technology Stack

* Kotlin
* Android Studio
* Jetpack Compose

---

## 🎯 Project Objective

The objective of this project is to design and develop a functional Android application that enhances user engagement while strengthening core Android development concepts such as UI design, state management, and user interaction.
